Грамматика

Определим грамматику для объявления структуры на языке Go G[‹DEF›] в нотации Хомского с продукциями P:

1) <DEF> → "let" <LET>
2) <LET> → letter <IDENTIFIER>
3) <IDENTIFIER> → letter <IDENTIFIER> | digit <IDENTIFIER>
4) <IDENTIFIER> → "=" <ASSIGNMENT>
5) <ASSIGNMENT> → "{" <OPEN>
6) <OPEN>→ ‘ " ’ <STRING>
7) <STRING> → letter <STRINGREM>
8) <STRINGREM> → letter <STRINGREM> | "_" <STRINGREM> | digit <STRINGREM>
9) <STRINGREM> → ‘ " ’ <ENDSTRING>
10) <ENDSTRING> → ":" <COLON>
11) <COLON> → digit <NUMBER>
12) <COLON> → ‘ " ’ <STRING2>
13) <STRING2> → letter <STRINGREM2>
14) <STRINGREM2> → letter <STRINGREM2> | "_" <STRINGREM2> | digit <STRINGREM2>
15) <STRINGREM2> → ‘ " ’ <ENDSTRING2>
16) <NUMBER> → digit <DIGIT>
17) <ENDSTRING2> → "," <OPEN>
18) <ENDSTRING2> → "}" <END>
19) <DIGIT> → "," <OPEN>
20) <DIGIT> → "}" <END>
21) <END> → ";"

letter → ‘a’ | ‘b’ | … | ‘z’ | ‘A’ | ‘B’ | … | ‘Z’
digit → ‘0’ | ‘1’ | … | ‘9’
Следуя введенному формальному определению грамматики, представим G[‹DEF›] ее составляющими:
Z = ‹DEF›;
VT = {a, b, c, ..., z, A, B, C, ..., Z, _, {, }, \n, \t, let, "};
VN = { <DEF>, <LET>, <IDENTIFIER >, < ASSIGNMENT >, <OPEN>, < STRING >, < STRINGREM >, <ENDSTRING>, <COLON>, <NUMBER>, <STRING2>, <STRINGREM2>,<ENDSTRING2>,<DIGIT>,<END>}.
