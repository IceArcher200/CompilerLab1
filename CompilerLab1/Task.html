Ассоциативный массив – это абстрактный тип данных, с помощью которого хранятся пары ключ-значение. данных, как переменная.

Для описания переменных в языке JavaScript используется служебное слово "let".

Формат записи:

let название_массива={ключ:значение};

Примеры:

Ассоциативный массив с целочисленным значением:

let arr = {"key":10};

Ассоциативный массив со строкой в качестве значения:

let newArr = {"Новосибирск":"Россия"};

Ассоциативный массив с несколькими записями:

let cities={"Омск":"55 регион","Кемерово":"42 регион", "Томск":"70 регион"};
